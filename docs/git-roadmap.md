# Git Roadmap

## Stage 1: Daily Git

Learn `status`, `add`, `commit`, `diff`, `log`, `show`, `restore`, `switch`, and `branch`. Your goal is to understand what is in the working tree, index, and repository before every commit.

## Stage 2: Team Git

Practice feature branches, merges, pull requests, remotes, upstream tracking, review comments, conflict resolution, and safe branch deletion.

## Stage 3: Recovery Git

Practice `restore`, `reset`, `revert`, `commit --amend`, and `reflog`. A senior developer should be able to recover from most local mistakes without panic.

## Stage 4: History Git

Practice rebase, interactive rebase, cherry-pick, tags, release branches, hotfix branches, and force-with-lease. Learn when rewriting history is acceptable and when it is not.

## Stage 5: Debugging And Internals

Practice `bisect`, `blame`, `grep`, object inspection, refs, reflog, and index concepts. These skills separate command memorization from real Git understanding.
